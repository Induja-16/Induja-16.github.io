Projects

Student Result Administration System Using RPA

A hybrid solution combining Robotic Process Automation (RPA) and web technologies to streamline result processing in educational institutions. Using UiPath, the system extracts student marks from PDF reports and automates Excel updates and uploads to an ERP-like system. The frontend was built with HTML, CSS, and JavaScript for intuitive result input and display, while Firebase powers real-time backend storage and authentication. This end-to-end automation reduces manual errors, speeds up workflows, and improves academic record management.

Fake Job Posting Detector

A machine learning-based system built to detect fraudulent job postings using NLP techniques and classification models. Processed and vectorized job ad text using TF-IDF and sentiment features, addressed class imbalance with SMOTE, and achieved 91% accuracy using Random Forest and strong recall with a fine-tuned BERT model. This project demonstrates how AI can enhance safety in online recruitment platforms.

Lush Green

An AI-powered web app that helps farmers make informed agricultural decisions by analyzing soil nutrients and detecting potential pest risks. We used machine learning models like Random Forest and SVM to predict optimal crops based on soil properties (e.g., potassium, calcium, pH) and to classify pest types from environmental data. LushGreen empowers smarter, data-driven farming to boost yield and sustainability.

Drug Bioactivity Prediction

An AI-driven solution to predict the IC50 value (half-maximal inhibitory concentration) of drug compounds, supporting faster and more accurate drug discovery. SMILES strings were converted into molecular descriptors using RDKit, and regression models such as Random Forest, XGBoost, and a deep neural network were trained. The best-performing model achieved R² of 0.89 and RMSE of 0.32, enabling reliable estimation of compound bioactivity. This project highlights how machine learning can streamline early-stage drug screening and reduce experimental costs..
